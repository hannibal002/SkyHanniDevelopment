Reports functions that handle SkyHanni events but are missing the @HandleEvent annotation, and flags functions annotated with @HandleEvent incorrectly.

A function is considered an event handler if its single parameter extends SkyHanniEvent, or if its name matches a @PrimaryFunction declaration on an event class. The function must also be public. 


// Missing annotation - will be flagged:
fun onSomething(event: SkyHanniEvent) { }

// Correct:
@HandleEvent
fun onSomething(event: SkyHanniEvent) { }

Also reports @HandleEvent on non-public functions, and on functions that do not take a SkyHanniEvent parameter and have no explicit eventType argument.
