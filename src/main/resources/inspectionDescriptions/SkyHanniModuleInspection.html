Reports SkyHanni object declarations that contain event handlers or repo patterns but are missing the @SkyHanniModule annotation.

Also flags class declarations annotated with @SkyHanniModule, since the annotation can only be applied to object declarations. 


// Missing annotation - will be flagged:
object MyModule {
    @HandleEvent
    fun onTick(event: SkyHanniTickEvent) { }
}

// Correct:
@SkyHanniModule
object MyModule {
    @HandleEvent
    fun onTick(event: SkyHanniTickEvent) { }
}

Use the quick fix to automatically add the @SkyHanniModule annotation.
